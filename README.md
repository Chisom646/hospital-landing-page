# HealthyCare Clinic Landing Page Template

A fully responsive and modern hospital/clinic landing page template.

## Features
- Fully responsive design
- Hero section with CTA button
- About, Services, Doctors, Testimonials, Contact sections
- Contact form included
- Modern and professional design
- Easy to customize

## How to Use
1. Open `index.html` in your browser to view the template.
2. Replace images in the `images/` folder with your own.
3. Edit text content directly in `index.html`.
4. Customize colors and styles in `css/style.css`.

## Free Images
All placeholder images can be replaced with your own. Free images were used for demonstration.
